# Lorem Ipsum Generator 🎨

A modern, ad-optimized Lorem Ipsum generator — text, emails, URLs, and domain
names, plus themed variants (Classic, Pirate, Tech, Startup). Built with Astro 5
to earn passive income through Google AdSense while ranking well organically;
secondary goal is funneling developer/founder traffic to
[Requiems API](https://requiems.xyz) and
[Bobadilla Tech](https://bobadilla.tech).

![Lorem Ipsum Generator](https://lorem-ipsum.bobadilla.tech/images/og-image.png)

🔗 **Live**: [lorem-ipsum.bobadilla.tech](https://lorem-ipsum.bobadilla.tech)

## Stack

- **Astro 5** (static output, ~0KB JS by default) — SEO + performance
- **TypeScript**, strict mode
- **pnpm** — always use pnpm, not npm/yarn
- **Cloudflare Pages** — hosting

## Quick start

```bash
pnpm install
pnpm dev   # http://localhost:4321
```
